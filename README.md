# OurWorld v0.0.0
A discord bot for the documentation of collaborative roleplays created by two Queer gamers who found themselves in fictional characters. Inspired by the Queer Worlds panel at QGCon 2023.

Designed by Amp and programmed by Avengineering.

*Formal readme pending.¯\_(ツ)_/¯*

### Current features: 
- Type /exportchannelmessages to get a .txt file of the entire channel.
- Compatible with nicknames,  for bots, and regular users!
- 

### Future(?) features: 
- Export based on timestamps/select start and end export messages.
- Export to html to open roleplays in browser in a little website format. Kind of like toyhouse, but for all your RP’s. 
- Sort roleplays with custom tags. 
- Also, it’s open source, so you can customize this however you’d like, forever!
